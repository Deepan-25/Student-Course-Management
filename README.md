# Student Course Management System

## How to Run

### Using Eclipse
1. Download or clone the repository.
2. Open Eclipse.
3. Click **File > Import > Existing Projects into Workspace**.
4. Select the project folder.
5. Run the `Main.java` file
A simple Java console application for managing students and courses.

## Features

1. Add Student
2. Add Course
3. Enroll Student in Course
4. Show Student Details
0. Exit

## Technologies Used

- Java
- Eclipse IDE
